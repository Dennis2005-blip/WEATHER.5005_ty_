 Project Description
The Weather Forecast Application is a mobile application developed to provide users with current weather information for selected locations. The app displays weather details such as temperature, weather conditions, and general forecasts.
This project was developed as part of the Mobile Application Development unit to demonstrate the use of APIs and real-time data in Android applications.
 Objectives
- To design a mobile application that displays weather information
- To consume and display data from a weather API
- To apply mobile application development concepts using Kotlin
 Features
- Display current weather conditions
- Show temperature and weather description
- Simple and user-friendly interface
- Runs on Android devices
 Technologies Used
- Kotlin
- Android Studio
- Weather API
- XML
